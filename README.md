# Sentiment Analysis with RNN, LSTM and BERT

## Overview

This project explores sentiment analysis on movie reviews using both classical and modern deep learning architectures.

The goal is to classify reviews as positive or negative and compare model performance across different NLP approaches.

## Technologies

- Python
- TensorFlow
- Keras
- Hugging Face Transformers
- DistilBERT
- Pandas
- NumPy

## Models

### Recurrent Neural Networks
- Simple RNN
- LSTM

### Transformer Models
- DistilBERT

## Workflow

1. Text Cleaning
2. Tokenization
3. Text Vectorization
4. Model Training
5. Model Evaluation
6. Performance Comparison

## Dataset

IMDB Movie Review Dataset
